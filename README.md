mongoImportExport
===============

Introduction
------------

Scripts de simplification des imports et des exports Mongo.
Note: pour l'instant l'import et l'export ne sont pas opérationnels, l'intérêt de ces scripts c'est la génération des lignes de commandes à jouer.

Présentation: 
------------

* mongoExport.py: Script qui génère la ligne de commande pour l'export d'une collection
* mongoImport.py: Script qui génère la ou les lignes de commande pour l'import de données dans une collection
